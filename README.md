# Docker Image for Legacy Apps #
This repo provide an legacy configuration to run the major Legacy Apps available

## OS Build ##
- Ubuntu 12.04.5 LTS

## Inside the Image ##
- Apache 2.2.22
- PHP 5.3.10

### Apache Modules ###
- libapache2-mod-php5

- php5-cli

- php5-json

- php-mail-mimedecode
- php-net-url

- php-soap
